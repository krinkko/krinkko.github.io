Better AutoDesk Deployment

18.03.2024
  • Software Deployment
  • AutoDesk
  • Sysadmin

Es ist wieder so weit für einen kleinen meinungsschwangeren Tutorialartikel zum Thema “AutoDesk für Sysadmins”. Wir wissen alle, dass nach der Einführung von AutoDesk ODIS Modern Installer irgendwie alles verschlimmbessert (neuDeutsch: Enshittyfied) wurde.

Zunächst kann man bei AutoDesk im Manage Portal zwei stabile Installationsmethoden erstellen: Install oder Deployment.

Install bietet den Typischen installer, der zwar mit dem Parameter -q bedient werden kann, was sich aber nur auf den Download der Daten auswirkt. Nimmt man den Parameter also mit, so wird nur der Download abgeschlossen. Der Installer hört danach auf. Für ein Deployment also nicht so geeignet. ABER, später hierzu mehr.
